# ZeroLeaks

## Overview

ZeroLeaks is a secure and privacy-focused platform for whistleblowers and journalists to anonymously share sensitive information. Built on the Sui blockchain with zero-knowledge proofs, ZeroLeaks ensures the authenticity of leaked documents while preserving the anonymity of sources. The documents are stored on walrus with encryption through seal more can be seen in the docs
